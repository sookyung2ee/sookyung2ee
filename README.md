# ⚾두산베어스 팬들을 위한 직관/집관 기록 웹앱 "BEARS10th"

<!--이미지추가-->
- 배포 URL: [bears10th.vercel.app](https://bears10th.vercel.app)
## BEARS10th 소개
- 두산베어스 팬을 위한 직관·집관 기록 플랫폼입니다. 경기를 직접 보러 가거나 집에서 응원한 기록을 남길 수 있습니다.
- 두산베어스의 경기 일정을 볼 수 있으며 종료된 경기의 스코어를 확인할 수 있습니다.
- 직관/집관 내역을 기록할 수 있으며 각각의 승률의 확인할 수 있습니다.
- 가고싶은 경기를 위시리스트에 추가할 수 있습니다.

## 📌 페이지별 기능

### 🏠 홈
- 오늘 날짜 기준으로 지난 경기와 다음 경기 일정을 보여줍니다.
- 홈화면의 하단에는 구단의 유튜브 영상 리스트가 연동되어 있으며, 영상을 클릭하면 해당 영상을 유튜브에서 볼 수 있습니다.
- 로그아웃 상태에서는 말풍선을 통해 로그인을 유도합니다.
- 회원가입도 가능하지만 간단하게 **게스트 모드**로 로그인도 가능합니다.

### 📊 기록
- 직관 기록, 집관 기록, 티켓 기록을 확인할 수 있습니다.
- 직관 기록 탭에서 직관 내역을 기록하면 자동으로 티켓이 추가됩니다.
- 직관/집관 스탬프, 티켓을 클릭하면 상세내역을 볼 수 있고 수정 및 삭제도 가능합니다.
- 상세내역 모달에서 수정 버튼을 눌러 기록을 수정할 수 있습니다.
- 직관/집관 기록에는 게임이 중복으로 입력될 수 없습니다.
- 직관 기록
  - 날짜, 메모, 좌석, 야구푸드를 기록할 수 있고, 날짜와 좌석은 필수 입력내용입니다.
  - 홈 경기는 흰색 유니폼, 원정 경기는 네이비 유니폼으로 스탬프가 찍힙니다.
  - 유니폼을 클릭하면 입력한 내용을 확인할 수 있습니다.
- 집관 기록
  - 날짜, 메모, 시청 디바이를 기록할 수 있고, 날짜와 시청 디바이스는 필수 입력내용입니다.
  - 스탬프를 클릭하면 입력한 내용을 확인할 수 있습니다.
- 티켓 기록
  - 캐릭터가 입은 유니폼의 색상(흰색/네이비)으로 경기의 홈/어웨이를 구분하였습니다.
  - 'more'버튼을 눌러 입력한 내용을 확인할 수 있습니다.

## 🛠️ 개발 환경
- **Front** : React, Vite, CSS Modules
- **Back-end** : Firebase Firestore, Firebase Authentication
- **버전 관리** : Git, GitHub
- **배포** : Vercel

## 🚀 추후 추가 예정 기능
- 📈 기록 및 승률 통계 
  - 월별 / 구장별 / 상대팀별 승률 분석 기능 추가
  - 직관에서 사용한 디바이스 통계 추가

  - 공유 받은 경기 수락/거절 기능
